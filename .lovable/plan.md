

# Landing Page Profesional para Jorge Flores

## Descripción General
Una página web elegante estilo "Dark Luxury" que servirá como CV online y herramienta para atraer pacientes a las prácticas clínicas de Jorge Flores, estudiante de odontología.

---

## Diseño Visual
- **Fondo:** Negro mate (#121212)
- **Acentos:** Verde oliva elegante (#6B8E23)
- **Texto:** Blanco
- **Estilo:** Minimalista con líneas limpias y formas geométricas abstractas (sin iconos dentales genéricos)
